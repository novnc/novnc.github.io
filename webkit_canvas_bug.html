This page uses the rendering engine from noVNC to demonstrate a Canvas rendering bug in WebKit. This bug was introduced between WebKit revision 66240 and 66396. The noVNC bug is here.

When you press Start, the noVNC engine will playback the beginning of a VNC/RFB session. You should see a complete 1024x768 Ubuntu desktop rendered below. If you have the bug only the first few rows of pixels will render. Note that the canvas operations have succeeded case, but they are simply not rendering correctly; if you resize the browser slightly, the entire desktop will immediately render.
